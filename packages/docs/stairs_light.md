# Stair Lights Automation

## Introduction
This is an automation to control the lights in the stairway.

Requirements:
* `scene` "Stairs TV On": This scene is applied when the TV is on
* `scene` "Stairs Motion": This scene is applied when motion is detected
* `binary_sensor` "Motion sensor": Used to detect motion on the stairs
* `media_player` "TV state": Get the state of the TV
